Gestion de parkings en C
Description
Ce programme en langage C permet de gérer des parkings à l’aide de fichiers CSV.
Il permet notamment de consulter les informations des parkings, vérifier les places disponibles et gérer la sortie des véhicules avec calcul du tarif.

Fonctionnalités
Consulter un parking ou la liste complète

Vérifier le nombre de places disponibles

Gérer la sortie d’un véhicule

Calculer le prix du stationnement (3€/heure)

Enregistrer les informations des clients

Fichiers utilisés
parking-metropole.csv : données des parkings

Vehicule.csv : véhicules stationnés

clients.csv : historique des clients

Compilation
gcc main.c -o main
Exécution
./main
Accès
Un code administrateur est demandé au lancement :
7658

Remarques
Le nombre de parkings et de véhicules est limité

Le programme dépend d’un format CSV strict
